# **App Name**: Chatter Anon

## Core Features:

- Anonymous Chat: Initiate a one-on-one random video/text chat using WebRTC with another anonymous user.
- Keyword Matching: Allow users to enter keywords of interest for optional matching with others sharing similar interests.
- AI Moderation: Employ real-time AI using the Google Vision API to detect and moderate NSFW content in chats.
- User Reporting: Enable users to report inappropriate behavior, storing these reports for review.
- Real-time Translation: Provide real-time translation of chat messages using the Google Translate API to facilitate cross-lingual communication, employing a tool which analyzes conversation context to improve translation relevance.

## Style Guidelines:

- Primary color: Soft purple (#A098E1) to convey a sense of calm and anonymity.
- Background color: Light gray (#F0F2F5), offering a neutral backdrop to keep the focus on the chat content.
- Accent color: Muted teal (#79A3B1) for buttons and interactive elements, providing a subtle contrast.
- Clean, sans-serif font for readability.
- Simple, outlined icons for key actions.
- A straightforward, single-panel layout.
